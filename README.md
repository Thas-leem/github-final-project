# Simple Interest Calculator

This project calculates **Simple Interest** using a simple formula.

## Formula

Simple Interest = (Principal × Rate × Time) / 100

## Inputs
- Principal amount
- Rate of interest
- Time period

## Output
- Simple interest value

## How to Run

### Bash Script
Run the script:

./simple-interest.sh

### Example
If Principal = 1000  
Rate = 5%  
Time = 2 years  

Simple Interest = 100
